# Grim Finance (\$GRIM)

## Overview

Official frontend repo of [grim.finance](https://app.grim.finance). Multichain yield farming optimizer.

## Env

Required env vars:

```
REACT_APP_NETWORK_ID = 250 // Network ID of FANTOM
```

## To Run

```
yarn add all
yarn start
```

The app will be running at http://localhost:3000/
